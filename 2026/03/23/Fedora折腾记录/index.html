Fedora折腾记录

本文最后更新于 2026年3月24日 晚上

折腾 Linux 系统的过程，本质上是一场关于“控制欲”与“审美”的修行。而在众多的发行版中，Fedora 始终占据着一个非常独特的位置：它不像 Ubuntu 那样保守，也不像 Arch 那样狂野。

如果要把 Linux 发行版比作汽车，Ubuntu 是大街上稳重的丰田，Arch 是需要你自己组装的越野车，而 Fedora 更像是一辆装满了各种原型技术的特斯拉——它总是最先搭载最新的引擎和自动驾驶系统。

🚀 Fedora 简介：站在创新的风口浪尖

Fedora 是由 Fedora Project 社区开发，并由红帽（Red Hat）赞助的发行版。它最大的标签就是**“激进”与“纯粹”**。

激进（Cutting Edge）： 你总能在 Fedora 上第一时间体验到最新的 Linux 内核、最新的 GNOME 桌面环境以及像 Wayland、PipeWire、Btrfs 这样引领行业标准的新技术。

纯粹（Freedom）： Fedora 严格遵守开源协议，默认不包含任何闭源驱动或多媒体解码器。这意味着“折腾”的第一步，通常就是亲手打破这些限制。

Gnome相关

GNOME Tweaks 是一款用于个性化桌面体验的工具，它允许用户自定义默认字体、窗口管理、工作区等多个方面。

安装官方的拓展管理应用

安装完成后，你可以在应用程序菜单中找到 “扩展” (Extensions) 图标。它是管理已安装插件的“开关”，支持开启、关闭以及进入插件的详细设置。

如何获取与安装新扩展

除了通过 DNF 安装部分系统级扩展外，最灵活的方式是使用浏览器插件：

访问官网：打开 https://extensions.gnome.org。

安装集成组件：浏览器会提示安装“GNOME Shell integration”扩展，点击允许并安装。

开启插件：在网页中找到心仪的扩展，直接点击右上角的 ON/OFF 开关即可自动下载并启用。

必装扩展推荐

针对追求高效、美观与系统监控的用户，以下扩展值得尝试：

  1. 桌面布局与效率类

​ Dash to Dock：将顶部的收藏栏（Dash）转换为类似 macOS 的固定 Dock 栏，支持自动隐藏。

​ Dash to Panel：将顶栏与 Dock 合二为一，模仿 Windows 任务栏布局，非常适合习惯单底栏操作的用户。

​ Just Perfection：GNOME 界的“瑞士军刀”。它可以让你隐藏几乎任何不需要的 UI 元素（如搜索框、顶栏时钟、辅助功能图标等），实现极致纯净的桌面。

  1. 美化与 UI 增强类

Blur my Shell：为 GNOME 的活动概览界面、面板和 Dash 添加精美的毛玻璃（高斯模糊）效果，瞬间提升桌面高级感。

User Themes：允许从用户目录下加载自定义的 Shell 主题，是深度定制皮肤的必备前提。

AppIndicator and KStatusNotifierItem Support：让托盘图标（如 Discord、网易云音乐、Telegram）重新出现在顶栏右侧，解决 GNOME 默认不显示第三方应用图标的问题。

Coverflow Alt-Tab: 改进 Alt+Tab 切换窗口的视觉效果，采用类似macOS的Coverflow立体翻转样式。

compiz-alike-magic-lamp-effect: 为窗口最小化添加类似Compiz的“魔灯”缩放动画效果。

  1. 功能增强类
    Clipboard Indicator：在顶栏添加剪贴板历史记录。对于开发者和文字工作者来说，这是大幅提升生产力的工具。

​ Vitals：在顶栏实时监控 CPU 温度、频率、内存使用率、风扇转速以及网络速度。它的 UI 设计非常简洁，不会显得杂乱。

​ Caffeine ：关闭屏幕保护程序和自动挂起

安装软件

命令行安装软件

  • curl: 命令行数据传输工具。支持 HTTP、FTP 等多种协议，常用于下载文件、调用 API 接口或排查网络连通性。

  • git: 目前最流行的分布式版本控制系统。无论是管理自己的代码脚本，还是从 GitHub 克隆项目，它都是核心工具。

  • btop: 一个颜值极高的终端资源监视器。它比传统的 top 或 htop 更直观，能以图表形式实时展示 CPU、内存、网络和磁盘的使用情况。

  • vim: 极其强大的终端文本编辑器。虽然有一定学习曲线，但在服务器环境或快速修改配置文件时，它的效率极高。

  • vlc: 全能的多媒体播放器。几乎能解码所有格式的视频和音频。在命令行安装后，它也提供 cvlc 这种无界面运行模式。

  • gimp: 开源的图像处理程序（类似 Photoshop）。它是一个功能齐全的 GUI 软件，适合进行照片编辑、绘图或图标制作。

  • fastfetch: 系统信息展示工具（neofetch 的现代化替代品）。运行它会用漂亮的 ASCII 字符展示你的操作系统图标以及硬件配置信息。

  1. Remmina (org.remmina.Remmina)
    功能：远程桌面客户端

用途：如果你需要从 Linux 电脑远程连接到 Windows 桌面（使用 RDP 协议）或者其他 Linux 服务器（使用 VNC 或 SSH），Remmina 是首选工具。

特点：支持多窗口管理、文件传输和剪贴板共享。对于系统管理员来说，它是管理远程服务器集群的利器。

  1. Kdenlive (org.kde.kdenlive)
    功能：专业级非线性视频编辑器

用途：它是 Linux 生态中最强大的开源视频剪辑软件之一，类似于 Adobe Premiere 或剪映。

特点：支持多轨道编辑、丰富的转场效果、关键帧动画以及 4K 视频导出。它基于 KDE 框架，非常稳定且功能全面。

  1. Steam (com.valvesoftware.Steam)
    功能：游戏分发平台

用途：全球最大的数字游戏发行平台。

特点：通过内置的 Proton (WINE) 技术，Steam 使得成千上万的原生 Windows 游戏可以完美运行在 Linux 系统上。如果你想在 Linux 上玩 3A 大作，这是必装软件。

  1. OBS Studio (com.obsproject.Studio)
    功能：视频录制与直播推流软件

用途：屏幕录制、网课制作或在 Bilibili/YouTube 等平台进行直播。

特点：支持多源输入（窗口捕获、摄像头、麦克风等），可以进行实时混音和场景切换。它是目前全球主播和录屏用户的行业标准工具。
